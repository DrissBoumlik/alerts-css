Install

NPM

$ npm i alerts-css -D

Yarn

$ yarn add alerts-css

jsdelivr

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/alerts-css@1.0.1/assets/css/alerts-css.min.css">
<script src="https://cdn.jsdelivr.net/npm/alerts-css@1.0.1/assets/js/alerts.min.js"></script>

Fixed alerts

  • Primeiramente você precisa usar nosso arquivo alerts.js.
<script src="https://cdn.jsdelivr.net/npm/alerts-css@1.0.1/assets/js/alerts.min.js"></script>
  • Você precisa adicionar o parametro "data-fade-time" com o valor em sugundos.
  • Type: Number
  • Ex: data-fade-time="3"
<button type="button" name="button" id="top-left">Top Left</button>
<div class="fixed-alerts fixed-alerts_top fixed-alerts_left">
                  
  <div class="alert alert_info alert_none alert-fade" id="alert-top-left" data-fade-time="3">
    <div class="alert--icon">
      <i class="fas fa-check-circle"></i>
    </div>
    <div class="alert--content">
      Atualizado com sucesso!
    </div>
    <div class="alert--close">
      <i class="far fa-times-circle"></i>
    </div>
  </div>
  
</div>
const btnTopLeft = document.getElementById('top-left')
const alertTopLeft = document.getElementById('alert-top-left');

btnTopLeft.addEventListener('click', function toggleAlert() {
  alertTopLeft.classList.toggle('alert_none')
})
